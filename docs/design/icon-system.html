PageMint — icon system

The mark is a printer's stamp: italic Fraunces P on cream paper, framed in ink, finished with a single red registration dot. Reads as a press, not an app tile. Survives down to 16px by dropping the dot and grain — the italic P is the load-bearing element.

01 · Master mark

The stamp

Four expressions. Use Paper as the canonical mark — it's what ships on the Chrome Web Store hero, the website, and the in-product wordmark. Ink (solid) is for stickers, swag, and avatars. Night is for dark surfaces. Mono is for press kits and Chrome's automatic monochrome favicon mode.

02 · Size system

Survives the toolbar

Chrome ships the toolbar icon at 16, 32, 48, and 128. The dot drops at 32 and below; the grain drops at 32 and below; the corner radius tightens proportionally. At 16px the icon is just the italic P in a thin-bordered square — and it still reads as PageMint, not as another rounded gradient blob.

06 · Production notes

What to actually ship

Files needed for the extension: 16, 32, 48, 128 PNG (light variant for default), plus a 128 PNG with the dark frame for the optional theme_icons manifest field — Chrome will swap based on the user's theme. Export as PNG with transparent background outside the rounded square.

Files needed for the store / site: the four master marks at 1024px PNG + SVG, the three lockups at 2x PNG + SVG, the 1280×800 store hero, and one 440×280 small tile.

Don't ship: a full-bleed gradient version, a monogram inside a circle, or a "P" with a download arrow through it. The italic P + frame + dot is the system. Resist embellishment.

Implementation tip: these mockups use Fraunces as a web font. For the actual icon PNGs, render once at 1024px in a vector tool (Figma / Affinity) and downsample — don't rely on browser font rendering at 16px, it's too soft. The 16px file should be hand-tweaked: full-pixel border, italic P sitting on the baseline with manual hinting.
