Why this works

  • Cream paper as default. Every AI extension is dark+glow. A warm light mode is an immediate differentiator — and on-brand for a tool literally named PageMint.
  • Serif wordmark. Fraunces (italic P, roman PageMint, tiny red dot) looks like a masthead, not a SaaS logo. Costs you nothing and carries the whole brand.
  • Black button, mint chip. Mint is used once, as the preset dot. The commit action is black ink. This reads as confident; emerald-CTA reads as template.
  • Monospace for specs only. IBM Plex Mono for "A4 · Portrait · 0.5″ · 100%" and the footer signal — the places where technical honesty matters.
  • One line of trust copy. "Saves locally · no uploads" replaces three collapsible sections. Same message, 3 words instead of 60.

What changed from today

  • Killed: PRESET / SAVED SETTINGS / DELIVERY metric card. Its info now lives in the header chip and the monospace specs line.
  • Killed: "Before you export", "Browser-print limits", "Local-first permissions" accordions. Move to the options page under "Help" if any survive at all.
  • Replaced: "Save this page as PDF / PageMint prepares the active tab..." → "Print this page & save it." with a tab-context card showing what you're saving.
  • Added: Tab context card with favicon + title + host. Users actually want confirmation of which page will be exported — the current popup omits this entirely.
  • Added: ⏎ affordance on the primary button. Pressing Return in the popup exports. Cheap power-user win Web to PDF doesn't have.
  • Added: Discreet Settings link in the footer so users can reach preferences without polluting the utility experience.

Copy set

  • Headline: "Print this page & save it." — active verb, no "PDF" (the button says PDF; redundant). The italic "& save it." adds voice.
  • Primary button: "Export" + ⏎. Single word. Not "Export current tab" — the tab card above already tells you which tab.
  • Footer: "Saves locally · no uploads" — replaces the whole Local-first permissions accordion.
  • Preset chip: "Exact" — one word, not "EXACT PDF". Preset names should be nouns the user can read like a word.
